# Echoly Roadmap

This document outlines the planned internal improvements and future feature additions for **Echoly**, categorized by their development phase.

## 🟢 Phase 1: Enhanced Core & Utilities (Upcoming)
- [ ] **Custom Hotkey Mapping**: Allow users to remap keyboard shortcuts for play/pause, speed, and manual advance.
- [x] **Rich Text Support (RTF)**: Basic support for highlighted text and bold/italic formatting within the prompter.
- [ ] **Advanced Timer Logic**: Display "Time Remaining" based on current scroll speed and remaining text length.
- [ ] **Auto-Save**: Ensure script edits in Editor Mode are saved locally every 30 seconds.


## 🟡 Phase 2: Professional Features & AI (Mid-Term)
- [ ] **Voice-Activated Scrolling**: Real-time speech-to-scroll using on-device Apple Speech recognition.
- [ ] **Webcam PiP Feedback**: A small, optional overlay of the user's webcam feed to help maintain framing and eye contact.
- [ ] **Smart Script Analysis**: AI-powered summaries and "speaking tone" suggestions for different script segments.
- [ ] **Multi-Display Mode**: Dedicated "Operator" and "Presenter" windows to support professional teleprompter glass setups.

## 🔴 Phase 3: Connectivity & Ecosystem (Long-Term)
- [ ] **Mobile Remote Controller**: Control prompter playback and speed from a smartphone via a local Wi-Fi web interface.
- [ ] **iCloud Sync**: Automatically sync scripts and preferences across multiple macOS devices.
- [ ] **Recording Integration**: Direct integration with system recording to start a video capture when the prompter begins.
- [ ] **Export Options**: Support for exporting scripts as `.docx` or formatted Markdown files.

---

## Completed Milestones 🏆
- **2.3.0**: Major architectural refactor and view decomposition.
- **2.2.0**: Added minimalist About window and MVVM implementation.
- **2.0.0**: UI modernization with Glassmorphism and Active Focus Mask.
- **1.2.0**: Professional Cue System (`[PAUSE]`, `[SLOW]`, `[CUE]`).
